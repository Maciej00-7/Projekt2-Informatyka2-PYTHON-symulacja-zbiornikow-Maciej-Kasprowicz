# Projekt2-Informatyka2-PYTHON-symulacja-zbiornikow-Maciej-Kasprowicz

W pliku projekt2_scad.py znajduje się główny projekt symulacyjny 4 zbiorników wody, przycisk start stop słóży do rozpoczęcia lub zatrzymania procesu przepływu wody
na końcu 2 zbiornika znajduje się zawór którego stan możemy ustawić przy użyciu listy.

W pilku test_projektu.py znajduje się kod testowy z wykorzystaniem biblioteki pytest, testuje on poprawne działanie niektórych funkcji symulacji.
